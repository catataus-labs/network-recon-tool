# Network Recon Tool 🛡️

**Developed by: Luiz Felipe (catataus-labs)**

Initial security project focused on network reconnaissance. This is the first of several programs I will develop and enhance during my 5-month intensive Cybersecurity roadmap (May - October 2026).

## 🚀 About the Project
I developed a simple Python script using the `socket` library and basic network services. The goal is to perform a TCP port scan to verify if a target is accessible.

* **Technical Learning:** Through this project, I deepened my understanding of the `connect_ex((ip, port))` function. I learned how Python handles low-level connection returns.
* **Flexibility:** The `target_ip` and `target_port` variables can be manually adjusted within the code to scan different targets and ports.
