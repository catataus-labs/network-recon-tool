# Network Recon Tool 🛡️

**Projeto desenvolvido por: Luiz Felipe (catataus-labs)**

Projeto inicial de segurança com foco em reconhecimento de rede. Este é o primeiro de uma série de programas que irei desenvolver e aprimorar durante meu plano intensivo de 5 meses em Cybersecurity (Maio - Outubro 2026).

## 🚀 Sobre o Projeto
Fiz um código simples em Python utilizando a biblioteca `socket` e serviços básicos de rede. O objetivo é realizar um escaneamento de porta TCP para verificar se um alvo está acessível.

* **Aprendizado técnico:** Nesse projeto, desenvolvi meu conhecimento sobre a função `connect_ex((ip, port))`. Compreendi como o Python lida com o retorno de conexões em baixo nível.
* **Flexibilidade:** O `target_ip` e o `target_port` podem ser alterados manualmente no código para escanear diferentes alvos e portas.

## 🗺️ Roadmap de Desenvolvimento
